# auto-bloging-
Automated blogging portal built with Node.js Express, supporting dynamic admin panel (password `9696`) and Firebase Firestore database integration (with local JSON fallback).

## Features
- **Modern Glassmorphic UI**: Beautiful dark mode and responsive layouts for both public viewing and admin control.
- **Dual Database Mode**: Seamless connection to Firebase Firestore. Fallback to local JSON database (`posts.json`) if Firebase key is missing.
- **Admin Panel**: Add and delete blogs securely.
- **Docker-ready / Cloud-ready**: Environment variables configured.
